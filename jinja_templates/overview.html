Overview

This is a Django web application running on a customized Amazon Web Services AMI.

It's designed to demonstrate some of the capabilities of the AMI, which has been tailored for geospatial applications.

It's a great starting point if you are creating a backend for a mobile web application that requires geospatial capabilities.

You are free to copy the AMI and use it for your Geospatial projects! It's all open source.
